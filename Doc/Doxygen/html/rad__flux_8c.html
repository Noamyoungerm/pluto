rad_flux.c File Reference

Compute the flux for the (M1) radiation transport equations. More...

#include "pluto.h"

Detailed Description

Compute the flux of the conservative radiation transport equations in the direction given by g_dir. This function defines the component of the hyperbolic flux tensor of the standard RHD equations.
In what follows:

  • VXn, MXn are the velocity, momentum components in the direction given by g_dir (normal, "n")
  • VXt, MXt and VXb, MXb are the transverse components (tangent "t" and bi-tangent "b").
